# Логотип

Положите сюда файл логотипа под именем `logo.png` (или `logo.svg` —
поменяйте расширение в `c.JupyterHub.logo_file` в `jupyterhub_config.py`
и в volume в `docker-compose.yml`).

Сам файл логотипа в этот репозиторий не входит — добавьте его сами
(в `.gitignore` он не попадает намеренно, чтобы не решать за вас,
можно ли хранить корпоративный бренд-актив в этом гит-репозитории;
при желании исключите `branding/logo.png` через `.gitignore` сами).

Где применяется: страница входа, шапка Hub, страница выбора/спавна
сервера (`c.JupyterHub.logo_file`). Внутри самого интерфейса JupyterLab
(вкладка с ноутбуками) логотип этим механизмом не меняется — это
отдельная, более объёмная кастомизация через labextension/тему, если
понадобится — скажите.

Рекомендуемый размер — как у стандартного логотипа Jupyter: примерно
250×50px (широкий, невысокий), PNG с прозрачным фоном.
